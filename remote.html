Remote Support

Click the appropriate button below to launch the remote assistance tool.

Instructions

  1. Find your operating system listed below.
  2. Click the corresponding logo.
  3. A file will be downloaded to your computer and you will receive a prompt asking if you want to 'Save' or 'Run' the file. Please click 'Run'.
  4. Be sure to click 'Yes' if you see a popup asking for your permission to run the application.

Windows

This includes all Windows desktops or laptops, and many Windows tablets. As long as your tablet is running full Windows and not Windows RT, this button will work for you. It does not include Windows Phones.

OS X

This includes all Apple computers such as Macbooks, Mac Pros, iMacs, and Mac Minis. It does not include Apple devices like iPhones, iPads, or iPods.

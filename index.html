About Me

As a software engineering student in the final year of my undergraduate studies, I take pride in crafting quality software using varying technologies. Object Oriented design is where I have the most experience, and I have worked in low-level languages such as C++, C#, Ada, and Java, as well as scripting languages such as Ruby and Python. Although my primary working environment is Windows, I like spending time in CLIs and have made an effort to familiarize myself with Debian-based distributions of Linux. Building software or products "from the ground up" is enticing, but I recognize that an efficient developer uses code and frameworks that have already been created to craft something significant in less time. Consistency is something I value highly, be it in UI, class design, or something else entirely, as consistency helps keep surprise to a minimum. I use the knowledge I have to create valuable products, but more importantly to continue learning in my constant endeavor to be better.
